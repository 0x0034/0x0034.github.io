    k8s核心数据结构

    November 26, 2020
    kubernetes
    k8s, kubernetes

    核心数据结构 # Group,Version,Resource 核心数据结构详情如下 资源组数据结构字段说明如下: Name : 资源组名称 Version: 资源组下所支持的资源版本 PerferredVersio: 当一个资源组存在多个资源版本时,首选版本 资源版本数据结构字段说明如下: Versions: 所支持的资源版本列表 资源数据结构字段说明如下: Name: 资源名称 SingularName: 资源的单数名称. 这个必须由小写字母组成,默认使用资源种类(Kind) 的小写形式进行命名. Namespaced: 资源是否拥有所属命名空间 Group: 资源所所在的资源组名称 Version: 资源所在的资源版本 Kind: 资源种类 Verbs: 资源可操作的方法列表 ShortNames: 资源的简称,例如pod简称为po
